# 👤 Professional Profile Card UI

ဒီ Project ဟာ **HTML5** နဲ့ **CSS3** ကိုသာ အသုံးပြုပြီး ဖန်တီးထားတဲ့ သေသပ်လှပပြီး ခေတ်မီတဲ့ **Profile Card** ဒီဇိုင်းတစ်ခုဖြစ်ပါတယ်။ Developer portfolio သို့မဟုတ် Social platform တွေမှာ အသုံးပြုဖို့ ရည်ရွယ်ပြီး တည်ဆောက်ထားတာ ဖြစ်ပါတယ်။

---

## 🌟 Features (ထူးခြားချက်များ)

* **Clean & Modern Design:** ရိုးရှင်းပြီး ဆွဲဆောင်မှုရှိတဲ့ Blue & White theme ကို အသုံးပြုထားခြင်း။
* **Social Media Integration:** Facebook, Twitter, Instagram နဲ့ YouTube icons လေးများကို သေသပ်စွာ ထည့်သွင်းထားခြင်း။
* **Interactive Buttons:** Follow နဲ့ Message buttons များတွင် စိတ်ဝင်စားစရာကောင်းသော Hover effects များ ပါဝင်ခြင်း။
* **Engagement Stats:** Followers, Comments နဲ့ Shares အရေအတွက်ကို Icons လေးများဖြင့် ဖော်ပြထားခြင်း။
* **Fully Responsive:** မည်သည့် Device တွင်မဆို ပုံစံမပျက်ဘဲ အဆင်ပြေစွာ မြင်တွေ့နိုင်ခြင်း။

## 🛠️ Built With (အသုံးပြုထားသော နည်းပညာများ)

* **HTML5** - ကတ်၏ တည်ဆောက်ပုံ (Structure) အတွက်။
* **CSS3** - Flexbox layout၊ Border radius၊ Shadows နှင့် Styling များအတွက်။

## 🚀 How to Run (အသုံးပြုပုံ)

1.  ဒီ Repository ကို **Clone** လုပ်ပါ သို့မဟုတ် **Download ZIP** ဆွဲပါ။
2.  Folder ထဲရှိ `index.html` ဖိုင်ကို Browser ဖြင့် ဖွင့်လှစ်၍ ကြည့်ရှုနိုင်ပါသည်။
